# HOUSE-TRACKER
# House Championship Portal

A real-time house championship tracking system built for a 1,000+ student school, 
using Firebase Realtime Database for instant multi-device sync — house points update 
live across every device without refreshing.

## Features
- Live scoreboard synced across devices via Firebase
- Online/offline status indicator
- House-wise point tracking and management
- Responsive dashboard UI

## Tech Stack
- HTML/CSS/JavaScript
- Firebase Realtime Database

## Why I built this
Our school's house point system was being tracked manually across spreadsheets and notice boards, which meant results were always delayed and inconsistent between houses. I built this to give every house real-time visibility into standings the moment points were updated — no more waiting for someone to manually update a board.
